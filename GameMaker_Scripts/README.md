# GameMaker Integration Guide

This folder contains the GML scripts needed to build your **Energy-Aware Command Center**.

## 0. Create Project
**Recommended Location:** Create a new folder named `GameMaker_Project` inside this repository root.
1.  Open GameMaker.
2.  Click **New** -> **Game**.
3.  Save it as `EnergyDashboard` inside the `GameMaker_Project` folder you just created.

## 1. Import Scripts
1.  Open your GameMaker Project.
2.  Create a new Script asset called `script_DataLoader`.
3.  Copy the contents of `script_DataLoader.gml` into it.
4.  Create a new Script asset called `script_GraphRenderer`.
5.  Copy the contents of `script_GraphRenderer.gml` into it.

## 2. Setup the Object
1.  Create a new Object `obj_EnergyDashboard`.
2.  **Create Event:**
    ```gml
    load_energy_data();
    ```
3.  **Draw Event:**
    ```gml
    draw_energy_graph(50, 50, 800, 400);
    
    // Simple Controls
    draw_text(50, 500, "Press LEFT/RIGHT to switch runs");
    ```
4.  **Key Press - Left:**
    ```gml
    if (variable_global_exists("current_run_index") && global.current_run_index > 0) {
        global.current_run_index--;
    }
    ```
5.  **Key Press - Right:**
    ```gml
    if (variable_global_exists("current_run_index") && global.current_run_index < array_length(global.energy_data.runs) - 1) {
        global.current_run_index++;
    }
    ```

## 3. Add Data
1.  Copy `data/gamemaker_export.json` (generated by Python) into your GameMaker project's `Included Files` (datafiles) section.

## 4. Run
Launch the game. You should see the graph!
